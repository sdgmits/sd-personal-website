# Certification Logos

Place your certification logo images in this folder.

## Recommended Format
- **File format:** PNG with transparent background
- **Dimensions:** 200x200px or square ratio
- **File naming:** Use descriptive names matching your data.ts references

## Example Files
- azure-solutions-architect.png
- power-platform-architect.png
- dynamics-365.png

Update the `logo` field in your certifications array in `src/utils/data.ts` to match the filename.
